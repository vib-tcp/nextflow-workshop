# Data folder

This folder contains some example data files to be used in the exercises.

- `crocodile_dataset.csv`: a small comma-separated file containing some information about different crocodile species. (fetched from [here](https://www.kaggle.com/datasets/zadafiyabhrami/global-crocodile-species-dataset))
- `boardgames.csv`: a small comma-separated file containing some information about different board games. (fetched from [here](https://www.kaggle.com/datasets/chik0di/board-games-dataset-complete-features))
